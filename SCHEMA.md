# Схема статьи (контракт)

Каждая статья — это один `.md` файл в папке `articles/`. Файл состоит из двух частей: **frontmatter** (метаданные между `---`) и **тело** (сам текст в Markdown).

## Frontmatter

```yaml
---
title: "Заголовок статьи (H1)"
slug: "url-statyi-translitom"
seo_title: "SEO-заголовок до 60 символов"
seo_description: "Мета-описание до 160 символов, попадает в RankMath."
focus_keyword: "главный ключ"
category: "Овощи и зелень"
tags:
  - тля
  - вредители
status: draft
images:
  - https://mir-doma.pro/wp-content/uploads/2026/07/tlya-1.jpg
---
```

| Поле | Обязательное | Что делает |
|------|--------------|------------|
| `title` | да | Заголовок поста (H1). Без него файл пропускается. |
| `slug` | нет | URL статьи. Если нет — строится из title. |
| `seo_title` | желательно | Title для RankMath. Лимит — 60 символов. |
| `seo_description` | желательно | Description для RankMath. Лимит — 160 символов. |
| `focus_keyword` | нет | Фокусный ключ RankMath. |
| `category` | да | Должна совпадать с существующей категорией сайта. |
| `tags` | нет | Список меток. |
| `status` | нет | `draft` или `publish`. Переопределяет настройку плагина. |
| `images` | нет | Зарезервировано. Пока картинки вставляются прямо в тело. |

## Тело

Обычный Markdown. Поддерживается:

- `## Заголовок` → H2, `### Подзаголовок` → H3
- списки (маркированные и нумерованные)
- **жирный**, *курсив*, [ссылки](https://mir-doma.pro)
- таблицы
- картинки: `![alt-текст](https://...jpg)` — alt берётся в alt-тег
- цитаты `>` и блоки кода

Плагин превращает это в нативные блоки Gutenberg, так что в редакторе статья откроется обычными блоками, а не «Классикой».

Дача и участок: Ландшафтный дизайн · Водоемы и системы полива · Газоны и дорожки · Заборы и ограждения · Хозяйственные постройки
Дом и уют: Интерьер дачи · Системы отопления · Утепление и энергосбережение · Электрика и сантехника · Мебель своими руками
Строительство и ремонт: Строительство на участке · Кровля и фасады · Ремонт дачного дома · Внутренняя отделка · Инструменты и материалы
Заготовки и хранение: Консервация · Заморозка · Хранение урожая · Погреба и кладовые · Рецепты из своего урожая
